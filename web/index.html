1입장 — 포털 2개

  • 빌드 포털: catalog.workshops.aws/join + Event access code
  • 경기 포털: agentic-football.aws.dev + Team Code (별도 사이트)
  • 리전 us-east-1 어제와 다름 — 모델 액세스도 여기

2빌드 4단계 (CloudShell + Harness, 노코드)

  1. 워크숍 내용 훑기 — 규칙·구조. "1단계: 팀 구축 – 샘플 코드 받기"까지
  2. CloudShell 열기 → "단계1: 저장소 클론" 4줄 명령 → 샘플 코드 클론
  3. 전술도구 배포 — "3단계 … AgentCore Gateway"의 "5개 Agent 배포 7(5+2)" 명령 → 람다+게이트웨이 자동 생성
  4. 하네스로 Agent 5개 생성 — 프롬프트 붙여넣기 + 모델 선택 + 게이트웨이. 각 Agent의 ARN 확보
키로 IDE는 메인 아님 — 프롬프트/지시문 다듬는 선택적 보조로만 언급됨.

3경기 포털에 연결

  1. agentic-football.aws.dev → Team Code 입력
  2. "내 팀" 페이지에 5개 Harness ARN 붙여넣기
  3. 연습경기로 Agent 정상 응답 확인 → 다른 팀에 도전

4우리 키트(shfoot) — 브레인으로

  • 프롬프트: prompts/{gk,def,mid,fwd}.md → 하네스 Agent에 그대로 붙여넣기 (전원 공 쫓기 금지 = 승부처)
  • 정책 숫자: policy.py 상단 존 좌표·슛레인지·압박트리거 → 하네스/프롬프트에 반영
  • 스키마 검증 (CloudShell은 리눅스):
git clone https://github.com/nexusailabs/shfoot && cd shfoot
python3 reconcile.py          # 내장 샘플로 GREEN 확인
python3 reconcile.py obs.json # 샘플 관측으로 키/좌표계 검증

5성적 올리기

  • 모델: 포지션별 다르게(사고깊이 vs 응답속도). 가벼운 모델 기본, 무거운 건 여유 있을 때만
  • 포지션 협응: 단일 Agent 최적화 ✗ → 역할/존 규율로 협응
  • CloudWatch: 경기 중 Agent 결정 과정 로그 확인 = 디버깅
  • 세부 튜닝: 슛 강도·패스 방식·압박 타이밍 한 번에 하나씩

6경기 중 영어 감독 지시 라이브 레버

사이드라인에서 자연어 지시 → 영어만, 구체적으로. 증상에 맞는 줄 눌러 복사.
기본 (시작 시)

Defenders, hold your zones. Do NOT chase the ball out of position.

Only the single nearest player presses the ball. Everyone else keeps formation.

Forward, stay on the last defender's line. Shoot on sight when in range.

Goalkeeper, stay on your line unless the ball enters the penalty box.

Midfielder, prioritize forward passes into open space, not sideways.

뭉침(ball-chasing) 보일 때 ★

STOP swarming the ball. Return to your assigned zones immediately.

Defenders, stop pushing forward. Recover to your defensive line now.

지고 있을 때

We are behind. Push the defensive line up and commit one extra player to attack.

Midfielder, take more risks: dribble forward and shoot from range.

이기고 있을 때

We are ahead. Drop deeper, keep possession, and run down the clock.

Avoid risky passes. Clear the ball long when pressured.

상황별

Opponent is attacking down the left. Shift the defensive line to cover that flank.

Stamina is low. Reduce pressing, hold compact shape, conserve energy.

Forward, be more decisive. Shoot earlier when you have a clear lane.

Too many turnovers. Play safer short passes and keep possession first.
